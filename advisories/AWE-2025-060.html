National Instruments LabVIEW VI File Parsing Out-of-Bounds Read VulnerabilityAWE-2025-060

CVE ID CVE-2025-2633
Vendor National Instruments
Product LabVIEW
Affected version ≤ 2025 Q1
Patched version 2025 Q1 Patch 3
Summary The affected application contains an out-of-bounds read vulnerability while parsing specially crafted VI files. This could allow a local attacker to potentially execute arbitrary code in the context of the current process.
