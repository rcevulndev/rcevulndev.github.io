Schneider Electric EcoStruxure Power Build Rapsody SSD File Parsing Double Free VulnerabilityAWE-2026-001

CVE ID CVE-2025-13844
Vendor Schneider Electric
Product EcoStruxure Power Build Rapsody
Affected version ≤ 2.8.6
Patched version 2.8.6.200
Summary A double free vulnerability exists in EcoStruxure Power Build Rapsody when parsing SSD files. The issue can potentially be exploited to execute arbitrary code, but requires a legitimate user to open a malicious SSD file.
Resources ICSA-26-015-10
Timeline 13.05.2025: Report sent to CISA
14.01.2026: Public release of security advisory
