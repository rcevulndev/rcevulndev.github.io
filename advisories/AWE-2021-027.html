Horner Automation Cscape File Parsing Out-of-Bounds Read VulnerabilityAWE-2021-027

CVE ID CVE-2021-32975
Vendor Horner Automation
Product Cscape
Affected version ≤ v9.90 SP3.5 (v9.90.104)
Patched version v9.90 SP5
Summary The affected application lacks proper validation of user-supplied data when parsing fonts. This could lead to an out-of-bounds read. An attacker could leverage this vulnerability to execute code in the context of the current process.
