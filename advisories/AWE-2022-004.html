OMRON CX-Programmer CXP File Parsing Use-after-Free VulnerabilityAWE-2022-004

CVE ID CVE-2022-25230
Vendor OMRON
Product CX-Programmer
Affected version ≤ v9.76.1
Patched version v9.77
Summary A use-after-free vulnerability exists that could result in arbitrary code execution due to lack of input validation, when a malicious CXP file is opened with CX-Programmer.
