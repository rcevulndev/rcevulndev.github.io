mySCADA myPRO OS Command Injection VulnerabilityAWE-2022-039

CVE ID CVE-2022-0999
Vendor mySCADA
Product myPRO
Affected version ≤ v8.25.0
Patched version v8.26
Summary The application is vulnerable to two OS command injections within the API password functionality. An authenticated remote attacker can exploit these to inject arbitrary OS commands which will get executed in the context of NT AUTHORITY\SYSTEM.
