Delta Electronics DIAEnergie Multiple DLL Hijacking VulnerabilitiesAWE-2022-035

CVE Number -
Vendor Delta Electronics
Product DIAEnergie
Affected version < 1.08.02.004
Patched version 1.08.02.004
Summary The application is prone to multiple DLL hijacking vulnerabilities. Combined with the weak file and folder permissions (see CVE-2022-26839), a local low-privileged attacker can easily escalate privileges to NT AUTHORITY\SYSTEM.
