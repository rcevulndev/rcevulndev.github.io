Santesoft Sante DICOM Viewer Pro DCM File Parsing Stack-based Buffer Overflow VulnerabilityAWE-2023-117

CVE ID ​CVE-2023-35986
Vendor Santesoft
Product Sante DICOM Viewer Pro
Affected version ≤ v12.2.4
Patched version v12.2.6
Summary The affected application lacks proper validation of user-supplied data when parsing DICOM files. This could lead to a stack-based buffer overflow. An attacker could leverage this vulnerability to execute arbitrary code in the context of the current process.
Resources ICSMA-23-285-01
Timeline 27.06.2023: Report sent to CISA
12.10.2023: Public release of security advisory
