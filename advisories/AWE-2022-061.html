Horner Automation Cscape File Parsing Uninitialized Pointer VulnerabilityAWE-2022-061

CVE ID CVE-2022-3378
Vendor Horner Automation
Product Cscape
Affected version ≤ v9.90 SP7
Patched version v9.90 SP8
Summary The affected application lacks proper validation of user-supplied data when parsing fonts. This could lead to an out-of-bounds write via an uninitialized pointer. An attacker could leverage this vulnerability to potentially execute arbitrary code in the context of the current process.
Resources ICS-CERT/CISA Advisory
Timeline 03.06.2022: Report sent to ICS-CERT/CISA
04.10.2022: Public release of security advisory
