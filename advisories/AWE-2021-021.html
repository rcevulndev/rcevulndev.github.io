LCDS LAquis SCADA Multiple Reflected Cross-site Scripting VulnerabilitiesAWE-2021-021

CVE ID CVE-2021-32989
Vendor LCDS
Product LAquis SCADA
Affected version ≤ v4.3.1.1011
Patched version v4.3.1.1079
Summary The LAquis SCADA application is prone to a few reflected cross-site scripting vulnerabilities, which are collectively tracked under CVE-2021-32989. All of these can be triggered without authentication:

1) http://[IP]/<PAYLOAD>.lhtm
2) http://[IP]/GENER1_1.lhtml?ID=<PAYLOAD>
3) http://[IP]/relatorionome.lhtml?ID=a&TAG=<PAYLOAD>&NOME=logset1&TITULO=Changes%20LOG
