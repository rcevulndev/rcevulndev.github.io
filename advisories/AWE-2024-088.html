Fuji Electric V-SFT and TELLUS SIM2 File Parsing Multiple Stack-based Buffer OverflowAWE-2024-088

CVE ID CVE-2024-38309
Vendor Fuji Electric
Products V-SFT
TELLUS
TELLUS Lite
Affected versions ≤ v6.2.2.0
≤ v4.0.19.0
Patched versions ≤ 6.2.3.0
v4.0.20.0
Summary The specific flaws exist within the parsing of SIM2 files. The issues result from the lack of proper validation of user-supplied data, which can result in multiple stack-based buffer overflows. Local attackers are able to exploit these issues to potentially execute arbitrary code on affected installations of V-SFT and TELLUS. User interaction is required to exploit the vulnerabilities in that a user must open a malicious SIM2 file.
