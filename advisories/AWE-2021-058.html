Eurotherm by Schneider Electric GUIcon GD1 File Parsing Out-of-Bounds Read VulnerabilityAWE-2021-058

CVE ID CVE-2021-22809
Vendor Eurotherm by Schneider Electric
Product GUIcon
Affected version ≤ v2.0 (Build 683.003)
Patched version n/a
Summary An out-of-bounds read vulnerability exists that could result in disclosure of information and potentially arbitrary code execution due to lack of input validation, when a malicious GD1 file is opened with GUIcon.
