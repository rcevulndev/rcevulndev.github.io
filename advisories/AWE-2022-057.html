Elcomplus SmartICS Arbitrary Process Termination VulnerabilityAWE-2022-057

CVE Number CVE-2022-2088
Vendor Elcomplus
Product SmartICS
Affected version ≤ v2.3.4.0
Patched version v2.4
Summary The application supports the termination of certain processes related to SmartICS. However, the feature can be abused by an authenticated attacker to terminate any process running on the system that has SmartICS deployed, including services and applications that are being executed in the context of NT AUTHORITY\SYSTEM.
