Horner Automation Cscape EnvisionRV HMI File Parsing Out-of-Bounds Read Vulnerability AWE-2021-074

CVE ID CVE-2021-44462
Vendor Horner Automation
Product Cscape EnvisionRV
Affected version ≤ v4.50.3.1
Patched version v4.60
Summary The specific flaw exists within the parsing of HMI files. The issue results from the lack of proper validation of user-supplied data, which can result in a read past the end of an allocated data structure. Local attackers are able to exploit this issue to disclose information and to potentially execute arbitrary code on affected installations of Cscape EnvisionRV. User interaction is required to exploit the vulnerability in that a user must open a malicious HMI file.
