Products CX-One
Sysmac Studio
Affected versions ≤ v4.61.1
≤ v1.56
Patched versions April 2024 Update

Summary The affected applications contain a heap corruption vulnerability due to a misaligned free operation while parsing specially crafted CXP files. A local attacker could leverage this vulnerability to potentially execute arbitrary code in the context of the current process.
