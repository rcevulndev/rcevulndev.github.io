Schneider Electric EcoStruxure Power Build Rapsody SSD File Parsing Stack-based Buffer Overflow VulnerabilityAWE-2025-031

CVE ID CVE-2025-3916
Vendor Schneider Electric
Product EcoStruxure Power Build Rapsody
Affected version ≤ v2.7.12 FR
Patched version v2.8.1
Summary A stack-based buffer overflow vulnerability exists in EcoStruxure Power Build Rapsody when parsing SSD files. The issue can potentially be exploited to execute arbitrary code, but requires a legitimate user to open a malicious SSD file.
Resources SEVD-2025-133-03
