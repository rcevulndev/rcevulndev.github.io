mySCADA myDESIGNER Path Traversal VulnerabilityAWE-2021-062

CVE Number CVE-2021-43555
Vendor mySCADA
Product myDESIGNER
Affected version ≤ 8.20
Patched version 8.22
Summary The affected product fails to properly validate the contents of an imported project file, which can be exploited to perform path traversal attacks. This vulnerability may allow an attacker to plant files on the file system in arbitrary locations or overwrite existing files, potentially resulting in arbitrary code execution.
