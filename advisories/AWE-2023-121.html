OMRON CX-Designer XML External Entity Processing Information Disclosure VulnerabilityAWE-2023-121

CVE ID ​CVE-2023-43624
Vendor OMRON
Product CX-Designer
Affected version ≤ v3.740
Patched version v3.750
Summary The application suffers from an XML External Entity (XXE) injection vulnerability when a project file is loaded. Due to the improper restriction of XXE references, it is possible to exfiltrate arbitrary files from the file system in the context of the user under which the application is being executed.
