VISAM Automation Base (VBASE) XML External Entity Processing Information Disclosure VulnerabilityAWE-2021-040

CVE Number -
Vendor VISAM GmbH
Product VISAM Automation Base (VBASE) Server
Tested version v11.6.0.6
Patched version n/a
Summary VBASE suffers from an XML External Entity (XXE) processing vulnerability when parsing project files. This issue can be exploited to retrieve the content of arbitrary files from the file system, but requires user interaction (such as opening a malicious project file) to exploit.
