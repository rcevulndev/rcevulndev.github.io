OMRON CX-Programmer CXP File Parsing Out-of-Bounds Write VulnerabilityAWE-2022-095

CVE ID CVE-2022-43509
Vendor OMRON
Product CX-Programmer
Affected version ≤ v9.78
Patched version v9.79
Summary The specific flaw exists within the parsing of CXP files. The issue results from the lack of proper validation of user-supplied data, which can result in a write past the end of an allocated data structure. Local attackers are able to exploit this issue to potentially disclose information and to execute arbitrary code on affected installations of CX-Programmer. User interaction is required to exploit the vulnerability in that a user must open a malicious CXP file.
