Siemens JT2Go & Teamcenter Visualization CGM File Parsing Out-of-Bounds Read VulnerabilityAWE-2022-081

CVE ID CVE-2022-41662
Vendor Siemens
Products JT2Go
Teamcenter Visualization
Affected version ≤ 14.1.0.2
Patched version 14.1.0.4
Summary The affected applications lack proper validation of user-supplied data when parsing CGM files. This could lead to an out-of-bounds read. An attacker could leverage this vulnerability to execute arbitrary code in the context of the current process. User interaction is required to exploit the vulnerability in that a user must open a malicious CGM file.
Resources Siemens Security Advisory SSA-120378
Timeline 16.07.2022: Report sent to Siemens ProductCERT
08.11.2022: Public release of security advisory
