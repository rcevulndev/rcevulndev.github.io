Horner Automation Cscape File Parsing Heap-based Buffer Overflow VulnerabilityAWE-2022-049

CVE ID CVE-2022-30540
Vendor Horner Automation
Product Cscape
Affected version ≤ v9.90 SP5 (v9.90.196)
Patched version v9.90 SP6
Summary The affected application lacks proper validation of user-supplied data when parsing fonts. This could lead to a heap-based buffer overflow. An attacker could leverage this vulnerability to execute code in the context of the current process.
