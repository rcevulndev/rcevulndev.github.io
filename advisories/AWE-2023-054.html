Horner Automation Cscape CSP File Parsing Use-after-Free VulnerabilityAWE-2023-054

CVE ID CVE-2023-28653
Vendor Horner Automation
Product Cscape
Affected version ≤ 9.90 SP8
Patched version v9.90 SP9
Summary The affected application contains a use-after-free vulnerability that could be triggered while parsing specially crafted CSP files. An attacker could leverage this vulnerability to potentially execute arbitrary code in the context of the current process. User interaction is required to exploit the vulnerability in that a user must open a malicious CSP file.
Resources ICS-CERT/CISA Advisory ICSA-23-143-04
Timeline 19.12.2022: Report sent to ICS-CERT/CISA
23.05.2023: Public release of security advisory
