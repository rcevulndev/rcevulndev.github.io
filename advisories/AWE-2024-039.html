Rockwell Automation Arena Simulation DOE File Parsing Heap-based Buffer Overflow VulnerabilityAWE-2024-039

CVE ID CVE-2024-21913
Vendor Rockwell Automation
Product Arena Simulation
Affected version ≤ 16.20.03
Patched version v16.20.03
Summary The affected application contains a heap-based buffer overflow vulnerability while parsing specially crafted DOE files. A local attacker could leverage this vulnerability to potentially execute arbitrary code in the context of the current process.
