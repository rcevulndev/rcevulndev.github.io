SAP 3D Visual Enterprise Viewer GIF File Parsing Memory Corruption VulnerabilityAWE-2021-013

CVE ID CVE-2021-33659
Vendor SAP
Product 3D Visual Enterprise Viewer
Affected version ≤ v9.10.3.100000
Patched version Refer to SAP Security Note 3059999
Summary A memory corruption vulnerability exists that could result in disclosure of information or execution of arbitrary code due to lack of input validation, when a malicious GIF file is opened.
Resources SAP Security Note 3059999SAP Security Patch Day – June 2021
Timeline 09.04.2021: Report sent to SAP
09.04.2021: Tracking ID provided by SAP
14.04.2021: Vulnerability confirmed by SAP
08.06.2021: Coordinated public release
