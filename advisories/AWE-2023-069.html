Panasonic FPWin Pro File Parsing Stack-based Buffer Overflow VulnerabilityAWE-2023-069

CVE ID CVE-2023-28728
Vendor Panasonic
Product FPWin Pro
Affected version ≤ v7.6.0.3
Patched version v7.7.0.0
Summary The specific flaw exists within the parsing of PRO files. The issue results from the lack of proper validation of user-supplied data, which can result in a stack-based buffer overflow. Local attackers are able to exploit this issue to potentially execute arbitrary code on affected installations of FPWin Pro. User interaction is required to exploit the vulnerability in that a user must open a malicious PRO file.
