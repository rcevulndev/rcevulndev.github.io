Fuji Electric TELLUS V9 File Parsing Memory Corruption VulnerabilityAWE-2023-130

CVE ID CVE-2023-47580
Vendor Fuji Electric
Products TELLUS
TELLUS Lite
Affected version ≤ v4.0.17.0
Patched version v4.0.18.0
Summary The specific flaw exists within the parsing of V9 files. The issue results from the lack of proper validation of user-supplied data, which can result in a memory corruption. Local attackers are able to exploit this issue to potentially execute arbitrary code on affected installations of TELLUS and TELLUS Lite. User interaction is required to exploit the vulnerability in that a user must open a malicious V9 file.
