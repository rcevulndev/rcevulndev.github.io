Fuji Electric Alpha7 PC Loader File Parsing Stack-based Buffer Overflow VulnerabilityAWE-2022-056

CVE ID CVE-2022-1888
Vendor Fuji Electric
Product Alpha7 PC Loader
Affected version ≤ v1.6.6
Patched version n/a
Summary The affected application lacks proper validation of user-supplied data when parsing project files. This could lead to a stack-based buffer overflow. An attacker could leverage this vulnerability to potentially execute arbitrary code in the context of the current process.
