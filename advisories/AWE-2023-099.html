Hitachi EH-VIEW KBD File Parsing Out-of-Bounds Write VulnerabilityAWE-2023-099

CVE ID CVE-2023-3495
Vendor Hitachi
Product EH-VIEW
Affected version ≤ v6.10 (02) - Build 03
Patched version The product is end-of-life (EOL) and should not be used anymore.
Summary The affected application contains an out-of-bounds write vulnerability while parsing specially crafted KBD files. A local attacker could leverage this vulnerability to potentially execute arbitrary code in the context of the current process.
Resources hitachi-sec-2023-002
