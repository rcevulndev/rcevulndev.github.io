Siemens Tecnomatix Plant Simulation SPP File Parsing Memory Corruption VulnerabilityAWE-2023-104

CVE ID CVE-2023-41846
Vendor Siemens
Product Tecnomatix Plant Simulation
Affected version see references
Patched version see references
Summary The affected application contains a memory corruption vulnerability while parsing specially crafted SPP files. This issue could lead to various bugs, including out-of-bound reads and writes, as well as to an uninitialized pointer access. A local attacker could leverage this vulnerability to potentially execute arbitrary code in the context of the current process.
