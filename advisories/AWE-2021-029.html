Siemens Solid Edge Multiple XML External Entity Processing Information Disclosure VulnerabilitiesAWE-2021-029

CVE ID CVE-2021-37178
Vendor Siemens
Product Solid Edge
Affected version ≤ 221.00.00.114
Patched version SE2021MP7
Summary Solid Edge suffers from multiple XML External Entity (XXE) injection vulnerabilities, which are collectively tracked under CVE-2021-37178, when certain XML files are being parsed. Due to the improper restriction of XXE references, it is possible to exfiltrate arbitrary files from the file system in the context of the user under which the application is being executed.
