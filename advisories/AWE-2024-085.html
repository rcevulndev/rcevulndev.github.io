mySCADA myPRO Manager OS Command Injection VulnerabilityAWE-2024-085

CVE ID CVE-2024-47407
Vendor mySCADA
Products myPRO Manager
myPRO Runtime
Affected versions ≤ v1.2
≤ v9.2.0
Patched version v1.3
v9.2.1
Summary The application is vulnerable to an OS command injection vulnerability. An unauthenticated remote attacker can exploit this issue to inject arbitrary OS commands which will get executed in the context of root (WSL) or myscada9 (administrative user which is automatically added during installation of the product), depending on the injection point chosen.
