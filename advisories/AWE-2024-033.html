Delta Electronics DIAEnergie Path Traversal VulnerabilityAWE-2024-033

CVE ID CVE-2024-28171
Vendor Delta Electronics
Product DIAEnergie
Affected version ≤ v1.10
Patched version v1.10.00.005
Summary The application suffers from a path traversal vulnerability. An authenticated attacker can exploit the issue to place files in arbitrary locations on the file system. It is also possible to overwrite existing files.
