Elcomplus SmartPTT SCADA Server Path Traversal VulnerabilityAWE-2022-048

CVE ID CVE-2021-43930
Vendor Elcomplus
Product SmartPTT SCADA Server
Affected version ≤ v1.4
Patched version v2.3.4
Summary The application is vulnerable to path traversal attacks. This issue can be combined with CVE-2021-43934, to write arbitrary files to arbitrary locations on the file system.
