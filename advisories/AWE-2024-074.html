JTEKT ELECTRONICS Kostac PLC Programming Software KPP File Parsing Out-of-Bounds Write VulnerabilityAWE-2024-074

CVE ID CVE-2024-47134
Vendor JTEKT ELECTRONICS
Product Kostac PLC Programming Software
Affected version ≤ v1.6.14.0
Patched version 1.6.15.0
Summary The specific flaw exists within the parsing of KPP files. The issue results from the lack of proper validation of user-supplied data, which can result in a write past the end of an allocated data structure. Local attackers are able to exploit this issue to potentially execute arbitrary code on affected installations of KPP. User interaction is required to exploit the vulnerability in that a user must open a malicious KPP file.
Resources JPCERT/CC Advisory JVNVU#92808077
