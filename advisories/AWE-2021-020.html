Panasonic FPWIN Pro XML External Entity Processing Information Disclosure VulnerabilityAWE-2021-020

CVE Number CVE-2021-32972
Vendor Panasonic
Product FPWIN Pro
Affected version ≤ v7.5.1.1
Patched version v7.5.2.0
Summary FPWin Pro suffers from an XML External Entity (XXE) injection vulnerability when a project file is loaded. Due to the improper restriction of XXE references, it is possible to exfiltrate arbitrary files from the file system in the context of the user under which the application is being executed.
