Delta Electronics DIAEnergie Arbitrary File Upload VulnerabilityAWE-2021-031

CVE Number CVE-2021-32955
Vendor Delta Electronics
Product DIAEnergie
Affected version ≤ v1.7.5
Patched version n/a
Summary The affected product allows unrestricted file uploads, which may allow an attacker to remotely execute code.
