Fuji Electric Tellus Lite V-Simulator and V-Server Lite VPR File Parsing Out-of-Bounds Write VulnerabilityAWE-2021-046

CVE Number CVE-2021-38419
Vendor Fuji Electric
Product Tellus Lite V-Simulator and V-Server Lite
Affected version ≤ v4.0.11.0
Patched version v4.0.12.0
Summary The specific flaw exists within the parsing of VPR project files. The issue results from the lack of proper validation of user-supplied data, which can result in a write past the end of an allocated data structure. Local attackers are able to exploit this issue to execute arbitrary code on affected installations of Tellus Lite V-Simulator and V-Server Lite. User interaction is required to exploit the vulnerability in that a user must open a malicious VPR file.

Note: CVE-2021-38419 is assigned for an OOBW when parsing V8 files that was reported by kimiya via ZDI, however, ICS-CERT/CISA confirmed that CVE-2021-38419 also covers the vulnerability that I reported. It is not clear whether there was a bug collision, but since the vulnerabilities affect the parsing of different file types, it seems to be unlikely.
