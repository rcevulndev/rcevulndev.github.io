Delta Electronics CNCSoft DPB File Parsing Out-of-Bounds Read VulnerabilityAWE-2021-063

CVE ID CVE-2021-44768
Vendor Delta Electronics
Product CNCSoft
Affected version ≤ v1.01.30
Patched version v1.01.31
Summary The specific flaw exists within the parsing of DPB project files. The issue results from the lack of proper validation of user-supplied data, which can result in a read past the end of an allocated data structure. Local attackers are able to exploit this issue to potentially disclose sensitive information on affected installations of CNCSoft. User interaction is required to exploit the vulnerability in that a user must open a malicious DPB file.
