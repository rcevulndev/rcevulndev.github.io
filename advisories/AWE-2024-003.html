Siemens Solid Edge PAR File Parsing Uninitialized Pointer VulnerabilityAWE-2024-003

CVE ID CVE-2023-49131
Vendor Siemens
Product Solid Edge
Affected version ≤ v223.0 Update 9
Patched version v223.0 Update 10
Summary The affected application is vulnerable to uninitialized pointer access while parsing specially crafted PAR files. An attacker could leverage this vulnerability to execute arbitrary code in the context of the current process.
Resources Siemens Security Advisory SSA-589891
Timeline 02.08.2023: Report sent to Siemens ProductCERT
09.01.2024: Public release of security advisory
