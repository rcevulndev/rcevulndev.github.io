Santesoft Sante DICOM Viewer Pro DCM File Parsing Out-of-Bounds Read VulnerabilityAWE-2024-022

CVE ID CVE-2023-1453
Vendor Santesoft
Product Sante DICOM Viewer Pro
Affected version ≤ v14.0.3
Patched version v14.0.4
Summary The affected application lacks proper validation of user-supplied data when parsing DICOM files. This could lead to an out-of-bounds read. An attacker could leverage this vulnerability to potentially execute arbitrary code in the context of the current process.
Resources ICSMA-24-058-01
Timeline 17.12.2023: Report sent to CISA
17.02.2024: Public release of security advisory
