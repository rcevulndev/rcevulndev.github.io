Delta Electronics DIALink Insecure Folder PermissionsAWE-2021-043

CVE ID CVE-2021-38420
Vendor Delta Electronics
Product DIALink
Affected version ≤ v1.2.4.0
Patched version n/a
Summary The file and folder permissions that are being set for the application are weak and give extensive permissions to low-privileged, local user accounts. A local attacker can exploit this in multiple ways to escalate privileges to NT AUTHORITY\SYSTEM.
