Schneider Electric IGSS CGF File Parsing Heap-based Buffer Overflow VulnerabilityAWE-2021-008

CVE Number CVE-2021-22760
Vendor Schneider Electric
Product IGSS
Affected version ≤ v15.0.0.21140
Patched version v15.0.0.21141
Summary A heap-based buffer overflow vulnerability exists that could result in disclosure of information or execution of arbitrary code due to lack of input validation, when a malicious CGF (Configuration Group File) file is imported to IGSS Definition.
Resources Schneider Electric Security Notification
ICS-CERT/CISA Advisory
Timeline 12.04.2021: Report sent to Schneider Electric
13.04.2021: Schneider Electric provides acknowledgement and tracking ID
26.04.2021: Schneider Electric confirmed all reported security vulnerabilities
04.05.2021: Schneider Electric provides status update and patch schedule
07.05.2021: Schneider Electric provides status update and planned public security notification
08.06.2021: Coordinated public release
