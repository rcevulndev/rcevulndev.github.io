MicroDicom DICOM Viewer Stack-based Buffer Overflow VulnerabilityAWE-2024-062

CVE ID CVE-2024-28877
Vendor MicroDicom
Product DICOM Viewer
Affected version ≤ v2024.1
Patched version v2024.2
Summary The affected application lacks proper validation of user-supplied data. This could lead to a stack-based buffer overflow that can be triggered through the application's custom URL scheme. An attacker could leverage this vulnerability to potentially execute arbitrary code in the context of the current process.
Resources ICSMA-24-163-01
Timeline 01.03.2024: Report sent to CISA
11.06.2024: Public release of security advisory
