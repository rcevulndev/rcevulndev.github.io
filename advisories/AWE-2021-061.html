Advantech WebAccess HMI Designer PM3 File Parsing Use-after-Free VulnerabilityAWE-2021-061

CVE Number CVE-2021-42706
Vendor Advantech
Product WebAccess HMI Designer
Affected version ≤ 2.1.9.95
Patched version 2.1.11.0
Summary A use-after-free vulnerability exists that could result in arbitrary code execution due to lack of input validation, when a malicious PM3 file is opened with WebAccess HMI Designer.
