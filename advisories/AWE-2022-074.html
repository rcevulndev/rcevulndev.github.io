NEC EXPRESSCLUSTER X Insecure File and Folder PermissionsAWE-2022-074

CVE ID CVE-2022-34824
Vendor NEC
Products EXPRESSCLUSTER X
Affected version ≤ 5.0
Patched version Refer to NV22-014
Summary The file and folder permissions that are being set for the application when installed to a custom directory are weak and give extensive permissions to low-privileged, local user accounts. A local attacker can exploit this in multiple ways to escalate privileges to NT AUTHORITY\SYSTEM.
Resources NEC Security Advisory NV22-014
Timeline 14.05.2022: Report sent to NEC-PSIRT
04.11.2022: Public release of security advisory
