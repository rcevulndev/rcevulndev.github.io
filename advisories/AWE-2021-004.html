Schneider Electric IGSS CGF File Parsing Out-of-Bounds Read VulnerabilityAWE-2021-004

CVE Number CVE-2021-22756
Vendor Schneider Electric
Product IGSS
Affected version ≤ v15.0.0.21140
Patched version v15.0.0.21141
Summary An out-of-bounds read vulnerability exists that could result in disclosure of information or execution of arbitrary code due to lack of input validation, when a malicious CGF (Configuration Group File) file is imported to IGSS Definition.
Resources Schneider Electric Security Notification
ICS-CERT/CISA Advisory
