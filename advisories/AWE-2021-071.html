mySCADA myPRO OS Command Injection VulnerabilityAWE-2021-071

CVE ID CVE-2021-43984
Vendor mySCADA
Product myPRO
Affected version ≤ v8.20.0
Patched version v8.22.0
Summary The application is vulnerable to an OS command injection within the firmware update mechanism. An unauthenticated remote attacker can exploit this issue to inject arbitrary OS commands which will get executed in the context of NT AUTHORITY\SYSTEM on Windows systems and in the context of root on Linux systems.
