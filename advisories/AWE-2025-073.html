National Instruments Circuit Design Suite SYM Parsing Out-of-Bounds Read VulnerabilityAWE-2025-073

CVE ID CVE-2025-6034
Vendor National Instruments
Product Circuit Design Suite
Affected version ≤ v14.3.1
Patched version v14.3.2
Summary The affected application contains an out-of-bounds read vulnerability while parsing specially crafted SYM files. This could allow a local attacker to potentially disclose information and to execute arbitrary code in the context of the current process. However, user interaction is required in that a user must open a malicious SYM file.
Resources ICSA-25-273-06
Timeline 21.05.2025: Report sent to CISA
30.09.2025: Public release of security advisory
