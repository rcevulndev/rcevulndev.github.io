mySCADA myPRO Arbitrary File Upload VulnerabilityAWE-2021-024

CVE Number CVE-2021-33009
Vendor mySCADA
Product myPRO
Affected version ≤ v8.18.0
Patched version v8.20.0
Summary The application suffers from an arbitrary file upload vulnerability within the configuration settings. Combining this issue with CVE-2021-33013 (authentication bypass) and CVE-2021-33005 (path traversal), and considering the fact that the web server and services are being executed in the context of NT AUTHORITY\SYSTEM, a complete system compromise is trivial.
