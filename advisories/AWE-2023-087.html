OMRON CX-Programmer CXP File Parsing Use-after-Free VulnerabilityAWE-2023-087

CVE ID CVE-2023-38748
Vendor OMRON
Product CX-Programmer
Affected version ≤ v9.80
Patched version v9.81
Summary The specific flaw exists within the parsing of CXP files. The issue results from the lack of proper validation of user-supplied data, which can result in a use-after-free condition. Local attackers are able to exploit this issue to potentially execute arbitrary code on affected installations of CX-Programmer. User interaction is required to exploit the vulnerability in that a user must open a malicious CXP file.
