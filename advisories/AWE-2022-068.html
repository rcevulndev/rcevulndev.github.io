Bentley Systems MicroStation Connect DGN File Parsing Out-of-Bounds Read VulnerabilityAWE-2022-068

CVE ID CVE-2022-41613
Vendor Bentley Systems
Products MicroStation Connect
Affected version ≤ v10.17.0.209
Patched version v10.17.1
Summary An out-of-bounds read vulnerability exists that could result in disclosure of information or execution of arbitrary code due to lack of input validation, when a malicious DGN file is opened.
Resources ICS-CERT/CISA Advisory ICSA-22-293-01
Timeline 25.07.2022: Report sent to ICS-CERT/CISA
20.10.2022: Public release of security advisory
