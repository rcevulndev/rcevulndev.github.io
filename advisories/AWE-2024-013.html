Horner Automation Cscape CSP File Parsing Stack-based Buffer Overflow VulnerabilityAWE-2024-013

CVE ID CVE-2023-7206
Vendor Horner Automation
Product Cscape
Affected version ≤ v9.90 SP10
Patched version v9.90 SP11
Summary The affected application contains a stack-based buffer overflow vulnerability that could be triggered while parsing specially crafted CSP files. This could allow an attacker to execute arbitrary code in the context of the current process.
