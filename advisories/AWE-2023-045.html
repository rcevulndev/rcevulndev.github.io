OMRON CX-Drive SDD File Parsing Heap-based Buffer Overflow VulnerabilityAWE-2023-045

CVE ID CVE-2023-27385
Vendor OMRON
Product CX-Drive
Affected version ≤ v3.01
Patched version n/a
Summary The affected application is vulnerable to a heap-based buffer overflow while parsing specially crafted SDD files. An attacker could leverage this vulnerability to potentially execute arbitrary code in the context of the current process.
