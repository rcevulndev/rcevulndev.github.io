Siemens Parasolid & Solid Edge XT File Parsing Out-of-Bounds Read VulnerabilityAWE-2024-019

CVE ID CVE-2023-49125
Vendor Siemens
Products Parasolid
Solid Edge
Affected versions Refer to the vendor advisory
Patched versions Refer to the vendor advisory
Summary The affected applications contain an out-of-bounds read past the end of an allocated structure while parsing specially crafted TX files. This could allow an attacker to execute arbitrary code in the context of the current process.
