AGG Software Advanced Serial Data Logger Web Server Plugin Stored Cross-site Scripting VulnerabilityAWE-2021-017

CVE Number CVE-2021-32962
Vendor AGG Software
Product Advanced Serial Data Logger Web Server Plugin
Affected version ≤ v4.0.40.1014
Patched version v4.0.42 Build 512
Summary The Web Server plugin for Advanced Serial Data Logger is vulnerable to stored cross-site scripting attacks. A remote attacker can inject JavaScript code that will get stored on the server and executed every time the affected component is accessed by a user.
