OMRON CX-Programmer CXP File Parsing Stack-based Buffer Overflow VulnerabilityAWE-2022-094

CVE ID CVE-2022-43667
Vendor OMRON
Product CX-Programmer
Affected version ≤ v9.78
Patched version v9.79
Summary The affected application contains a stack-based buffer overflow vulnerability that could be triggered while parsing specially crafted CXP files. This could allow an attacker to potentially execute arbitrary code in the context of the current process.
